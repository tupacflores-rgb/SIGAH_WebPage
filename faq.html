Sobre SIGAH

SIGAH es el Sistema Integral de Gestión de Asistencias y Horarios, desarrollado por Rolando Maximo Tupac Flores Arce como proyecto de la materia Diseño UI/UX en la EET N°3100 "Rep. de la India".

Permite a instituciones educativas registrar asistencias, gestionar horarios de clases, generar reportes y enviar notificaciones automáticas a través del módulo SANE.

Backend  : Python 3.x
Base de datos: MariaDB (producción) / SQLite (desarrollo)
Frontend : HTML5 + CSS3 + JavaScript
Sistema de notificaciones: SANE (Python)

No utiliza frameworks externos pesados: el frontend es 100% HTML, CSS y JS nativo.

  • Inicio de sesión — Acceso seguro con roles
  • Dashboard — Estadísticas y accesos rápidos
  • Registro de alumnos — Alta y gestión de estudiantes
  • Asistencia — Toma y seguimiento por curso y materia
  • Horario — Grilla semanal interactiva
  • Reportes — Gráficos y exportación PDF
  • Notificaciones — Alertas en tiempo real (SANE)
  • Configuración — Gestión del sistema y usuarios

SANE es el Sistema Automático de Notificaciones Escolares, un módulo complementario a SIGAH también desarrollado en Python.

Se encarga de detectar automáticamente situaciones de riesgo (como alto ausentismo) y enviar notificaciones a padres, docentes y administradores sin intervención manual.

Uso del sistema SIGAH

Desde el Dashboard, accedé al módulo Asistencia. Seleccioná el curso, la fecha y la materia, luego marcá el estado de cada alumno y presioná Guardar Asistencia.

Desde el módulo Reportes, seleccioná el período y el curso. El sistema genera el gráfico y la tabla automáticamente. Podés exportar en PDF con el botón Exportar PDF.

El módulo SANE detecta automáticamente cuando un alumno supera el umbral. Se genera una alerta en Notificaciones y se envía aviso al administrador y/o docente.

El umbral se configura desde Configuración → Notificaciones.
