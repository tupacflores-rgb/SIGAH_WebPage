🟦 Módulo 1 — Fundamentos de la Web

1.1 HTML5: Estructura y semántica

El lenguaje base de toda página web. Aprendé a estructurar contenido con etiquetas semánticas.

📌 Temas principales:

  • Estructura básica: DOCTYPE, html, head, body
  • Encabezados: <h1> a <h6> y su jerarquía
  • Párrafos, saltos de línea: <p>, <br>
  • Código y texto preformateado: <pre>, <code>
  • Formularios: input, select, textarea, button
  • Tablas: thead, tbody, tr, th, td
✏️ Práctica recomendada:

Creá una página HTML que muestre tu currículum usando todos los niveles de encabezados y formularios.

💻 Ejemplo de código:

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>Título principal</h1>
    <p>Este es un párrafo.</p>
  </body>
</html>

1.2 CSS3: Estilos y diseño responsive

Dá vida visual a tus páginas. Desde colores y tipografías hasta layouts con Flexbox y Grid.

📌 Temas principales:

  • Selectores, clases e IDs
  • Variables CSS (:root, var())
  • Flexbox y CSS Grid
  • Media Queries — diseño responsive
  • Transiciones y animaciones
✏️ Práctica:

Diseñá la interfaz del sistema SIGAH respetando la paleta de colores y los componentes vistos en clase.

🟨 Módulo 2 — JavaScript y PHP

2.1 JavaScript — Interactividad del lado del cliente

Hacé que tus páginas respondan a acciones del usuario sin recargar.

📌 Temas principales:

  • Variables, tipos de datos, funciones
  • Manipulación del DOM
  • Eventos: onclick, onchange, onsubmit
  • Fetch API — consumo de servicios REST
  • LocalStorage para persistencia simple

💻 Ejemplo:

// Cambiar estado de asistencia
function toggleEstado(btn) {
  const estados = ['Presente','Ausente','Tardanza'];
  const idx = estados.indexOf(btn.textContent);
  btn.textContent = estados[(idx + 1) % 3];
}

2.2 PHP — Servidor web básico

Procesá formularios, conectate a bases de datos y generá HTML dinámico desde el servidor.

📌 Temas principales:

  • Variables, arrays, funciones
  • Formularios POST/GET
  • Conexión a MySQL con PDO
  • Sesiones y autenticación básica

🟩 Módulo 3 — Python para sistemas

3.1 Python base y automatización

El lenguaje principal de SIGAH y SANE. Desde la sintaxis básica hasta módulos de automatización.

📌 Temas principales:

  • Sintaxis, variables, listas, diccionarios
  • Funciones, clases y módulos
  • Manejo de archivos y errores
  • Bibliotecas: os, datetime, json, requests

💻 Ejemplo:

import datetime

def registrar_asistencia(alumno, estado):
    fecha = datetime.date.today()
    registro = {
        "alumno": alumno,
        "estado": estado,
        "fecha": str(fecha)
    }
    print(f"[{fecha}] {alumno}: {estado}")
    return registro

registrar_asistencia("Flores Arce, Rolando", "Presente")

3.2 Bases de datos: MariaDB, MySQL y SQLite

Almacenamiento y consulta de datos para sistemas reales.

📌 Temas principales:

  • Modelado de datos y tablas
  • SQL: SELECT, INSERT, UPDATE, DELETE
  • Relaciones y JOIN
  • Conexión desde Python (mysql-connector, sqlite3)
  • Migraciones y respaldos
